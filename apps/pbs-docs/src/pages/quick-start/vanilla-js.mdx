# Quick Start Guide

## Installation

### Using NPM

```shell copy
npm install @plainsheet/plain-bottom-sheet-core
```

### Using CDN

```html copy
<script src="https://cdn.jsdelivr.net/npm/@plainsheet/plain-bottom-sheet-core@latest/dist/plain-bottom-sheet-core.umd.js"></script>

// Or, via unpkg
<script src="https://unpkg.com/@plainsheet/plain-bottom-sheet-core@latest/dist/plain-bottom-sheet-core.umd.js"></script>
```

## Getting Started

### Using NPM

```ts copy
import { createBottomSheet } from "@plainsheet/plain-bottom-sheet-core";

const bottomSheet = createBottomSheet({
  content: `<div> 
        Your content
        </div>`,
});

bottomSheet.mount();
bottomSheet.open();

// Change the content, on the fly.
bottomSheet.props.content = `<div>New Content</div>`;
```

### Using CDN

```ts copy
const bottomSheet = window.PlainBottomSheetCore.createBottomSheet({
  content: ``,
});
// Rest is the same as NPM
```
